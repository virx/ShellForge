1. Install instructions
================================================
 - Extract package to folder, where you have writing permissions.
 - Execute file "install.sh" to create shortcut.
 - Execute file "sh" to run the program.

2. Program Description
================================================
 - Execute cunfigurable terminal command sequences with button click.
 - Command sequence buttons can be sectioned into different tabs.
 - Tab and command buttons are created according to config.txt file.

3. Development tasks
================================================
 - Configurable error recognition function.
 - Script running indicator.
